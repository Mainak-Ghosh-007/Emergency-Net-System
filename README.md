Emergency Net

Emergency Net is a robust full-stack web application designed to bridge the gap between citizens in distress and emergency services. Built using the Flask framework, this platform allows users to report emergency situations in real-time and locate critical services—such as police stations, hospitals, and ATMs—via an interactive map interface.
📌 Core Features
🚨 Emergency Reporting
Instant SOS: One-click functionality to report emergency situations.
Data Persistence: Securely stores emergency reports in a database for administrative review.
Real-time Management: Integrated admin panel to track, manage, and resolve active cases.
🗺️ Map & Location Services
Interactive Mapping: Integrated map interface for visual navigation and location pinning.
Proximity Search: Quickly find the nearest:
🚓 Police Stations
🚑 Hospitals / Ambulance Services
🏧 ATMs
Geospatial Assistance: Utilizes user location data to provide context-aware support.
🎨 UI/UX Design
Responsive Architecture: Built to work seamlessly across both desktop and mobile devices.
Emergency-First Interface: Minimalist and high-contrast design to ensure ease of navigation during high-stress situations.
🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript
Backend: Python (Flask)
Database: SQLite
Mapping: OpenStreetMap / Leaflet.js
